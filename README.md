# 🏴‍☠️ Onepiecedle

![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![One Piece](https://img.shields.io/badge/theme-One%20Piece-red)

**Onepiecedle** est un jeu de devinettes quotidien basé sur l'univers de *One Piece*. Inspiré par Wordle et Loldle, le but est de trouver le personnage mystère du jour en utilisant des indices basés sur leurs attributs (Équipage, Fruit du démon, Haki, Prime, etc.).

---

## 🎮 Comment jouer ?

Tapez le nom d'un personnage de One Piece pour commencer. Des indices apparaîtront pour vous guider vers la bonne réponse :

| Indicateur | Signification |
| :--- | :--- |
| 🟩 **Vert** | Correspondance exacte (ex: Même équipage). |
| 🟥 **Rouge** | Pas de correspondance. |
| 🟧 **Orange** | Correspondance partielle (ex: La prime est supérieure/inférieure). |

### Attributs comparés :
* **Genre** (Masculin, Féminin, Autre)
* **Affiliation** (Pirates, Marines, Révolutionnaires...)
* **Type de Fruit du Démon** (Paramecia, Logia, Zoan, Aucun)
* **Haki** (Lequel est maîtrisé ?)
* **Provenance** (Pays natal)
* **Prime** (Indication plus ou moins élevée)

## 🛠️ Installation et Lancement

Pour lancer le projet localement sur votre machine :

### Pré-requis
* [Node.js](https://nodejs.org/) (version 16 ou supérieure)
* npm ou yarn

### Étapes

1. **Cloner le dépôt :**
   
   ```bash
   git clone [https://github.com/AdrienTheophile/Onepiecedle.git](https://github.com/AdrienTheophile/Onepiecedle.git)
   cd Onepiecedle 

2. **Installer les dépendances :**
   
   ```bash
   npm install

3. **Lancer le serveur :**
   
   ```bash
   ng serve --open
